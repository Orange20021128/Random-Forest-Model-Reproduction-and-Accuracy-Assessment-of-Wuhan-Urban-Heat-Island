本研究以武汉主城区30m分辨率遥感与地理数据为基础，采用随机森林回归模型探究土地利用景观格局对地表温度LST的影响。首先整合 8 类土地利用的覆盖率、连通度、复杂度、聚集度4 类景观特征（共 32 个因子），并纳入 DEM、SVF、距绿地距离、距道路距离 4 个控制变量，构建模型自变量；以 Landsat 8/9 夏季合成地表温度 LST 为因变量。 数据预处理完成栅格有效像素提取、坐标匹配与数据集构建后，采用K-Means 空间聚类 + 空间块交叉验证 (SBCV) 规避空间数据泄露，训练随机森林回归模型，并通过 R²、RMSE、MAE 开展模型精度评价；进一步对模型预测残差进行Global Moran’s I 空间自相关分析，检验模型对城市热环境空间格局的解释能力。
